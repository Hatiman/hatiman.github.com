Настройка D-link

Подкючение кабелей D-Link DIR 320/NRU


Компьютеры подключаем в LAN порты синего цвета, кабель с интернетом вставляем в WAN порт. Reset - кнопка сброса настроек. USB - порт для локального принтера.

Как зайти на D-Link DIR 300

В адресной строке браузера вводим 192.168.0.1 и жмём Enter(Если что Enter это кнопка на клавиатуре, которая находится над правой кнопкой Shift).

В строку User Name вводим admin и нажимаем кнопку Log In, пароль вводить не надо по умолчанию он пустой.

Восстановление прошивки D-link

Если ваш роутер вдруг завис при перепрошивке и перестал загружаться или вы хотите вернуть стандартную прошику вам поможет консоль восстановления.

  1. Зажимаем кнопку reset при этом роутер должен быть выключен, затем продолжая держать кнопку reset включаем питание. Через 20 секунд отпускаем кнопку.
  2. Выставляем на компьютере IP адрес 192.168.0.2 маска 255.255.255.0, витой парой подключаем компьютер к роутеру в любой порт LAN.
  3. Открываем браузер и в адресную строку вводим IP адрес 192.168.0.1 жмем Enter и видим консоль восстановления.

  1. Качаем прошивку с официального сайта D-link, нажимаем Обзор, указываем до неё путь и жмём кнопку Update Software.
  2. Ждём примерно 2 минуты, после чего обновляем страницу и видим веб-морду роутера.

Настройка шифрования Wi-Fi на D-Link DIR 320

Жмём SETUP выбираем Wireless Setup и далее на кнопку Manual Wireless Connection Setup.

Напротив строчки Enable убераем галочку. В поле Security Mode выбираем Enadle WPA/WPA2 Wireless Security (enchanced). Далее в строке Network Key вводим ключ из восьми символов и нажимаем Save Settings.
