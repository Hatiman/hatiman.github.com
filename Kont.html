Настройка сетевой карты
 

Настройк Windows XP

Пуск => Панель управления

В открывшимся окне выбираем "Сеть и подключение к интернету". Если нету такого пункта, то ищите "Сетевые подключения"

Находим "Сетевые подключения"

Правой кнопкой мыши по "Подключению по локальной сети" и выбираем пункт меню "Свойства".

Подключений по локальной сети может быть несколько. Ищите изображение с двумя синими мониторами и без красного крестика.

Выбираем Протокол интернета TCP/IP => Свойства

Выставляем получить автоматически и нажимаем кнопку "OK"

Настройка для Windows XP завершена!

Настройка Windows Vista

Найдите данную иконку внизу-справа возле часов и нажмите на нём. В открывшейся вкладке нажмите "Центр управления сетями и общим доступом"

Нажмите в левом меню "Управление сетевыми подключениями"

Правой кнопкой мыши по "Подключению по локальной сети" и выбираем пункт меню "Свойства".

Подключений по локальной сети может быть несколько. Ищите изображение с двумя синими мониторами и без красного крестика.

Найдите в открывшемся окне кнопку "Свойства" и нажмите её.

Выбираем Протокол интернета TCP/IP v4 => Свойства

Выставляем получить автоматически и наживает кнопку "OK"

Настройка для Windows Vista завершена!

Настройка Windows 7

Найдите данную иконку внизу-справа возле часов и нажмите на нём. В открывшейся вкладке нажмите "Центр управления сетями и общим доступом"

Нажмите в левом меню "Изменение параметров адаптера"

Правой кнопкой мыши по "Подключению по локальной сети" и выбираем пункт меню "Свойства".

Подключений по локальной сети может быть несколько. Ищите изображение с двумя синими мониторами и без красного крестика.

В открывшимся окне нажмите кнопку "Свойства", затем в новом окне выделите пункт (не снимая галочки) "Протокол интернета TCP/IP v4" и нажмите ниже кнопку свойства.

В новом окне выставьте получить атвоматически и нажмите кнопку ОК.

На этом настройка сетевой карты для дальнейшей настройки роутера на Windows 7 завершена!

Настройка сети в Mac OS X 10.5.x

Откройте Яблочное меню (Apple menu) и выберите пункт Системные настройки.

Выберите параметр Сеть (Network).

Нажимаем на выпадающий список Размещений, в появившемся меню необходимо выбрать пункт Редактировать размещения..

Нажмите на кнопку со знаком "+" и впишите название размещения. Лучше всего название вашего оператора "это для того что бы Вы знали какое подключение относится к вашему оператору".

В поле Конфигурация выберите пункт Вручную. В соответвующие поля впишите настройки из договора. После этого нажмите кнопку Применить.

На этом настройка сетевой карты для дальнейшей настройки роутера на Mac OS X 10.5.x завершена!

Настройка сети в MacOS X 10.4.x

Откройте Параметры системы (System Preferences).

Выберите параметр Сеть (Network).

Количество интерфейсов зависит только от модели компьютера. Выбираем необходимый интерфейс для настройки. Для нас – это Встроенный Ethernet (Built-In Ethernet).

На вкладке TCP/IP укажите способ настройки IP-адреса: Вручную (Manually) и в диалоговом окне укажите IP-адрес (IP Address). Необходимо также ввести данные о маске подсети (Subnet mask) , адрес шлюза по умолчанию (Router) и адрес сервера имен (DNS), которые вы найдете в договоре.Закройте диалоговое окно.

На этом настройка сетевой карты для дальнейшей настройки роутера на MacOS X 10.4.x завершена!
